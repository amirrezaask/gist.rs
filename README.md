# Gist Cli 
Simple CLI tool to manage your github gist.

## Usage 
### Add command
```bash
USAGE:
    gist_rs add [OPTIONS] --file <file>

OPTIONS:
    -d, --desc <desc>    description of the gist
    -f, --file <file>    name of file to add to gist.
    -h, --help           Print help information
```
